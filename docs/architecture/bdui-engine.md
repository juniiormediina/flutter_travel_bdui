# BDUI Engine

## Overview

The BDUI engine is responsible for transforming a JSON contract into a visual experience.

Flow:

JSON Screen

↓

Parser

↓


Models

↓

Registry

↓

Renderer

↓

UI

## Responsibilities

- Parse contract.
- Validate schema.
- Resolve component types.
- Render components.
- Execute actions.

## Supported Components

- Banner
- Title
- Button
- Flight Search
- Offer Card
- Destination Carousel

## Non-Responsibilities

- Business logic
- Authentication
- Analytics
- Persistence
